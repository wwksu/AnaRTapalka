# Hamster Tap - Telegram Mini App

Простая тапалка в стиле Hamster Kombat для Telegram.

## Установка

1. Установи зависимости:
```bash
pip install -r requirements.txt
```

2. Создай бота через @BotFather в Telegram и получи токен

3. Открой `bot.py` и замени:
   - `YOUR_BOT_TOKEN_HERE` на токен твоего бота
   - `YOUR_WEBAPP_URL_HERE` на URL где будет хоститься приложение

## Запуск

```bash
python bot.py
```

Бот запустится на порту 8080. Для продакшена нужно будет:
- Настроить HTTPS (обязательно для Telegram WebApp)
- Использовать nginx или другой reverse proxy
- Поменять URL в коде на свой домен

## Структура проекта

- `bot.py` - Telegram бот и API сервер
- `index.html` - Главная страница приложения
- `style.css` - Стили
- `script.js` - Логика игры
- `image.jpg` - Картинка хомяка
- `users_data.json` - База данных пользователей (создаётся автоматически)

## Фичи

### Главный экран (Tap)
- Тап по хомяку +1 монета
- Анимация при тапе
- Счётчик энергии (1000 макс, восстановление 1/сек)

### Магазин (Shop)
- **Мульти-тап** - увеличивает монеты за тап (100 монет, х2 за уровень)
- **Энергия+** - увеличивает макс энергию на 500 (200 монет, х2 за уровень)
- **Авто-тап** - пассивный доход 1 монета/сек (500 монет, х2 за уровень)
- **Золотой скин** - золотая рамка вокруг хомяка (1000 монет, одноразово)

## Деплой

Для деплоя на сервер:
1. Залей файлы на сервер
2. Настрой HTTPS (Let's Encrypt)
3. Запусти бота через systemd или screen
4. Укажи правильный URL в bot.py

Можно использовать ngrok для тестирования локально.
